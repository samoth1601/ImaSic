Starting with the note "E" and going up...
F - darkness, anger, anguish
F# - suspense, mystery, mystical
G - sadness, resoluteness, weariness
G# - happiness, bright, cheerful
A - buoyancy, triumph, movement
A# - fear, tension, violence
B - cheerfulness, stability, simplicity
C - sadness, wistful, surprise
C# - longing, romance, sweetness
D - aggression, power, strength
D# - whimsy, nostalgia, desire
and back to "E" to start the pattern over
